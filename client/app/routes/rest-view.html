Collections Overview
Resource Description
Workflow Shared bioinformatics or biology workflow, contains an explainatory image, a description and possibly user-posted reviews.
Database Important databases for the domain of Oil & Gas, posted and reviewed by users, same as workflows.
Review Aforementioned reviews of the remaining collections
GET Method
Resource URI(s) Description
GET All /api/:collection Returns all documents in a collection
GET by id /api/:collection/:id Returns the document with inserted id in selected collection

GET Examples

Sample GET all request using cURL:

curl http://aleph.inesc-id.pt/vre/api/databases

Sample GET by id request using cURL:

curl http://aleph.inesc-id.pt/vre/api/databases/56a8fedafe23f6435c7b144d
POST Method
Resource URI(s) Description
Create documents /api/create/:collection Creates document in a collection
Edit by id /api/edit/:collection/:id Edits a document with inserted id in selected collection
Remember that schema must be respected

POST Examples

Sample POST Create document using cURL:

curl -H 'Content-Type: application/json' -X POST -d '{ "name":"Metagenomics Pipeline", "description":"FASTX, BLAST, MEGAN", "author":"user@domain.com" }' http://aleph.inesc-id.pt/vre/api/create/workflows

Sample POST Edit by id request using cURL:

curl -H 'Content-Type: application/json' -X POST -d '{ "name":"Metagenomics Pipeline", "description":"FASTX, BLAST, MEGAN, R", "author":"new.user@.new.domain.com" }' http://aleph.inesc-id.pt/vre/api/edit/workflows/56ca06c4624e59e10dcbe109
PUT Method
Resource URI(s) Description
Create Review /api/reviews/:collection/:id Creates review in a document with id :id
Remove Review /api/reviews/:collection/:docid/:revid Removes a review from a document in selected collection

PUT Examples

Sample PUT Create review using cURL:

curl -H 'Content-Type: application/json' -X PUT -d '{ "stars":"3", "body":"dont like this one very much...blast is very basic", "author":"someone@domain.com" }' http://aleph.inesc-id.pt/vre/api/reviews/workflows/56ca06c4624e59e10dcbe109

Sample PUT Remove review by id request using cURL:

curl -H 'Content-Type: application/json' -X PUT http://aleph.inesc-id.pt/vre/api/reviews/workflows/56ca05ee624e59e10dcbe108/56ca08f3624e59e10dcbe10a
DELETE Method
Resource URI(s) Description
Delete document /api/delete/:collection/:id Deletes document with id :id

DELETE Examples

Sample DELETE document using cURL:

curl -X DELETE http://aleph.inesc-id.pt/vre/api/delete/reviews/56ca091a624e59e10dcbe10c
